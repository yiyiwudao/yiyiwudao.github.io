### 3. 为什么会被封号？

以下是可能导致封号的常见原因：

1. **批量注册和 API 滥用**：使用脚本批量注册账号或过于频繁地调用 API。
2. **非法支付方式**：通过代付或使用黑卡进行支付。
3. **邮箱问题**：使用乱码邮箱或小众邮箱服务（如 Hotmail、Outlook）。
4. **IP 地址异常**：使用被广泛滥用的代理 IP 注册或登录。
5. **支付行为异常**：频繁更换支付卡片或支付失败过多次。
